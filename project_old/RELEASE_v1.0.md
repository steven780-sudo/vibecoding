# Chronos v1.0 发布说明

**发布日期**: 2025-10-21  
**版本**: MVP v1.0

---

## ✨ 核心功能

### 仓库管理
- 初始化文件夹为"时光库"
- 查看仓库状态和文件变更
- 支持中文文件名

### 快照管理
- 创建快照保存文件状态
- 选择要包含的文件
- 添加描述说明
- 查看完整历史记录

### 版本控制
- 查看历史快照时间线
- 展开查看详细信息
- 一键回滚到任意版本

### 分支管理
- 创建实验性分支
- 在分支间自由切换
- 合并分支到主版本
